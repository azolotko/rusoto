A map of one or more table names and, for each table, a map that describes one or more items to retrieve from that table. Each table name can be used only once per BatchGetItem request. Each element in the map of items to retrieve consists of the following: * ConsistentRead - If true, a strongly consistent read is used; if false (the default), an eventually consistent read is used. * ExpressionAttributeNames - One or more substitution tokens for attribute names in the ProjectionExpression parameter. Use the # character in an expression to dereference an attribute name.
